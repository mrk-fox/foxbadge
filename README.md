# foxbadge
My Fallout Badge (@Hack Club)

## Project Description
The project presented in this Repository is a versatile "Hacker-Badge" which has the following features:
1. Seeeduino XIAO ESP32-C6 as the main processor
2. A 4-mechanical switch (Cherry MX) controll
3. A 1.9 inch TFT screen
4. A battery system (NO BMS! ONLY BATTERIES WITH INTEGRATED BMS ARE SAFTE TO USE)
5. Supports 2 diffrent kind of lanyards
6. A case!
